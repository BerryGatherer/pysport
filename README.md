# SportOrg

### Packages

- PyQt5
- [sireader](https://pypi.python.org/pypi/sireader/1.0.1)
- [peewee](http://docs.peewee-orm.com/en/latest/peewee/quickstart.html)
- Flask
- jinja2

Install packages
```
pip install package-name

pip install sireader
```

or

```
pip install -t lib -r requirements.txt
```
